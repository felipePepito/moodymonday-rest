
## Description

Backend for the moody-monday App built with
[NestJS](https://github.com/nestjs/nest)

## Installation

```bash
$ npm install
```
Also configure file `/ormconfig.template.json` file to your needs and save it under `/ormconfig.json`\
Afterwards configure `/.dev.template.env` to your needs and save it under `/.dev.env`
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
